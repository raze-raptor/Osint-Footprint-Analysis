# Osint-Footprint-Analysis
OFA, Osint Footprint Analysis. This is a tool made in python to follow existing publicly accessible data to create a visible footprint of who youre looking at.


# How does it work?
It uses a database of known data breaches and connects somebodies socials to their IP address, usernames, MAC, and more. Using all of these details, the tool should be able to provide a comprehensive visual showing links between the inputted user details.
